# Selenium TestNG Automation Framework  
### A Complete Data-Driven Hybrid Framework (Selenium + Java + TestNG + Maven + POM)

This project is a real-world **UI Automation Framework** built using **Selenium WebDriver, Java, TestNG, Maven, and Page Object Model (POM)**.  
It includes **Data-Driven Testing**, **End-to-End Order Booking Flow**, and reusable utilities, following real industry standards.
